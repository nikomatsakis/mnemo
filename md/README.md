# Mnemo

Welcome to Mnemo, a language-learning co-pilot that blends human-curated goals with adaptive LLM-driven practice. These docs are “from the future”: they describe the experience we are building so we can stay aligned before we write code.

## Vision

- **Personal grammar roadmap.** Mnemo tracks the grammar concepts you want to master and keeps their decay curves visible.
- **Vocabulary in plain English.** You describe vocab groups (“foods you eat for breakfast”, “verbs for travel mishaps”) and Mnemo lets the LLM expand them into practice-ready sets.
- **LLM as instructor + judge.** One agent crafts prompts, another evaluates responses, offering retries, hints, or nudges when needed.
- **Progress you can feel.** A journal, timeline, and analytics view highlight what you’ve reinforced, what’s slipping, and what’s next.
