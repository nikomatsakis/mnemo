# User Guide Overview

Mnemo is designed for self-directed learners who want tight feedback loops without micromanaging flashcards. A typical learner journey looks like this:

1. **Create a study profile.** Pick your target language (Greek first), note your level, and set the cadence you want Mnemo to enforce.
2. **Describe grammar priorities.** Choose from the built-in concept catalog (cases, aspect, agreement, etc.) or add custom notes when you need nuance.
3. **Author vocab groups in plain English.** For example: “daily routine verbs”, “phrases for ordering coffee”, “Athenian street directions”. Mnemo lets the LLM fill in exemplars.
4. **Start a practice session.** Mnemo queue mixes grammar + vocab based on decay, then hands the baton to the generator agent.
5. **Respond + iterate.** You type your translation or fill-in answer. The judge agent can ask for clarification, offer a second try, or mark it with granular scores.
6. **Review insights.** Every attempt updates your decay curves and surfaces next-step recommendations. The journal auto-logs notable milestones or struggles (with your consent).

The rest of the User Guide drills into configuration and the live practice flow.
